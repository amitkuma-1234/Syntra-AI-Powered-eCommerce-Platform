## 🚀 What is Syntra?

**Syntra** is a next-generation AI-powered eCommerce platform designed to deliver a smarter shopping experience. Built on **Next.js** with SSR/SSG capabilities, Syntra combines intelligent product discovery with a clean, responsive UI — making it the perfect foundation for a modern online store.

> 💡 *Browse products → Get AI-powered recommendations → Add to cart → Checkout seamlessly*

---

## ✨ Features

- 🤖 **AI-Powered Recommendations** – Smart product suggestions based on user behavior
- 🛒 **Shopping Cart & Checkout** – Smooth cart management and checkout flow
- 🔍 **Product Search & Filtering** – Quickly find products by category, price, or keyword
- 🖼️ **Product Listings** – Rich product pages with images, descriptions, and pricing
- 👤 **User Authentication** – Secure user login and session management
- ⚡ **SSR / SSG** – Fast page loads with Next.js Server-Side & Static Generation
- 🎨 **Modern Responsive UI** – Clean design optimized for all screen sizes
- 🔤 **Geist Font** – Optimized typography via `next/font`
- 🌐 **Vercel Ready** – One-click deployment on Vercel

↓

↓

↓

## 🔧 Getting Started

1. **Clone the repository:**
```bash
   git clone https://github.com/amitkuma-1234/Syntra-AI-Powered-eCommerce-Platform.git
   cd Syntra-AI-Powered-eCommerce-Platform/shopebysnetch
```

2. **Install dependencies:**
```bash
   npm install
   # or
   yarn install
```

3. **Run the development server:**
```bash
   npm run dev
   # or
   yarn dev
```

4. **Open in browser:**
http://localhost:3000

> You can start editing by modifying `app/page.js` — the page auto-updates as you edit.
